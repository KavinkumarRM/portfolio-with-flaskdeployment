My Projects

Human Identifier

human identifier is a dynamically adaptable human identifier application which can be used for various realtime application such as human attendance tracking, automated photo transfer between friends and could be used as a security system

Poly Hyperparameter Tuning

Performing Grid search on all possible hyperparameter is time consuming and the most worst part is when you dont get the required result. This is one of my try to find out the optimal hyperparameter using the data aquired by grid search and Calculas.

Traffic Sign Detection

A completely functional real-time Traffic sign detection system capable of detecting traffic signs in low light evenings and nights. Implemented a custom-build traning system to deal with overly baised sign system in the United States

Musical Face

Being able to play music based on your face emotion would be of a great break through in music. This feature could be used in situation where you could not afford to loose a little distraction. Initially have made a spotify API with this feature for this project.

Binary tree classifier

Binary tree calssifier is a ML tree architecture consiting of ML models employed for multi level classification of NLP data. Each node consist of a ML mode to decide further flow of the data through the tree. Developed AutoML API for model selection to power the tree.

Flight crew identification

Build a computer vision system to ensure the minimum number of fight crew on board before taking off. Motion detection along with Res-Net served as the base for the algorithm which was trained to detect crew members from people passing by the camera
